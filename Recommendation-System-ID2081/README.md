# Friend-Based Page Recommendation

## Problem Statement

You are given the list of Facebook friendships and the list of Facebook pages followed by users.

Your task is to build a recommendation system that suggests pages to users based on pages followed by their friends.

For each Facebook user, recommend pages that:

- The user does **not** already follow
- At least one of their friends follows

Return the user ID and the page ID that should be recommended.
